# Screenshots Directory

Add your application screenshots here.

## Required Screenshots

### Web Application
- `web-dashboard.png` - Dashboard view with statistics and charts
- `web-visualization.png` - Data visualization page
- `web-upload.png` - CSV upload interface

### Desktop Application
- `desktop-login.png` - Login/Register dialog
- `desktop-dashboard.png` - Dashboard tab with charts
- `desktop-upload.png` - Upload tab interface
- `desktop-visualization.png` - Visualization tab with data table

## How to Add Screenshots

1. Take screenshots of your application
2. Save them with the exact filenames listed above
3. Place them in this `screenshots/` directory
4. The README.md will automatically display them

## Image Guidelines

- **Format**: PNG or JPG
- **Size**: Recommended 800-1200px width
- **Quality**: Clear and well-lit screenshots
- **File Size**: Keep under 1MB per image for faster loading
